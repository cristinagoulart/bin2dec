How to convert binary to decimal

For a binary number b with n digits:

bn ... b3 b2 b1 b0

The value of any binary number will be equal to the sum of its digits multiplied by their respective weights.

conversion = b0×20 + b1×21 + b2×22 + ... bn×2n

Example

Lets calculate decimal value of 1000002:

binary: 1 0 0 0 0 0
power of 2: 25 24 23 22 21 20 
              
                
Conversion  = 1⋅25+0⋅24+0⋅23+0⋅22+1⋅21+1⋅20 = 3210
